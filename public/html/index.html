Help
Controls

Zoom - Tap on canvas where you want to zoom in or use Q and E to zoom in/out.

x/y - Use WASD keys to move around the fractal.

Iterations - Use Z and X to increase/decrease number of iterations (detail).

Reset - Use R to reset the canvas.

Right click to save render.

Camera Settings

Dropdown on title - Select from fractals to display.

x/y - Controls camera position.

Iterations - Number of times to check if a value reached a critical escape condition, basically increases detail.

Exponent - Controls the d value of z = z^d + c.

Complex Real/Imaginary - Used as a fixed term for c values when displaying Julia Sets.

Render Settings

Iteration Coloring - Assigns a value based on iteration for each point.

Smooth Coloring - Assigns a color for each iteration using linear interpolation providing a smooth transition of colors between iterations.

Color Presets - Select predefined color schemes for the renders.

Gradient Loop - Makes color pallete loop from end to start.

RGB Factors/Shifts - RGB channels for adjusting a color pallete's factors or shifts depending on if gradient is looped.

Auto-render - On by default, won't render when settings change when off.
